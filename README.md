# CyberTrust
CyberTrust Is A RP2350A Based Security Key, it can be used as a FIDO2 Secure Key for secure authentication and account safety.

Features:
- Custom PCB
- Custom Enclosure
- Based on the RP2350A MCU
- WS2812B RGB LED
- User Presence Button
- USB-A Male Port
- OSH and OSS
